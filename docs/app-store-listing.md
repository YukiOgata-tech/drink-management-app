## 説明文

```
【このアプリについて】
Alcohol Log & Event Hubは、日々の飲酒量を記録し、振り返ることで健康的な飲酒習慣をサポートするアプリです。

飲酒を促進するアプリではありません。
自分の「適量」を知り、健康を意識した生活を送るためのツールです。

【主な機能】

■ かんたん飲酒記録
- ビール、ハイボール、サワー、日本酒、ワインなど豊富なプリセット
- バーコードスキャンで缶チューハイも簡単登録
- カスタムドリンクの追加も可能

■ 純アルコール量の自動計算
- 厚生労働省の基準に基づく適正量との比較
- 1日・1週間の摂取量を可視化

■ イベント機能（飲み会モード）
- 友人や同僚と一緒に記録
- QRコード・招待コードで簡単参加
- 飲み会後のランキング表示

■ 振り返り・統計
- 日別・週別・月別の飲酒傾向
- 休肝日のカウント

【こんな方におすすめ】
- 健康診断で肝臓の数値が気になる方
- 飲み過ぎを控えたい方
- 自分の飲酒パターンを把握したい方
- 飲み会の記録を残したい方

【ご注意】
- 本アプリは20歳以上の方を対象としています
- 飲酒運転は絶対にやめましょう
- 表示される数値は推定値であり、医療的判断の根拠にはなりません

適度な飲酒で、健康的な毎日を。
```
